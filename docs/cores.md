# Cores Helper

##### `set_var($key, $value)`
Fungsi ini digunakan untuk menyimpan data yang dapat diakses secara global.
```php
set_var('foo', 'bar');
```
##### `get_var($key, $default = '')`
Fungsi ini digunakan untuk mengakses data yang disimpan secara global.
```php
get_var('foo');   // bar
```
