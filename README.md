## Dokumentasi
### Konfigurasi
Konfigurasi aplikasi disimpan di dalam file `config.php`:

```php
return array(
  /**
  * Package name
  */
  'name' => 'londomloto/immortal',
  
  /**
  * Version
  */
  'version' => '1.0.0'
)
```
